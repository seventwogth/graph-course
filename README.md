### Задание 1: Рендеринг звезды из сэмпла (/StarTest)
**Цель:** Работа с геометрией и настройка базового рендера.
- [x] Взять пример отрисовки (DirectX / OpenGL / HLSL / GLSL).

- [x] Реализовать различные топологии:

### Задание 2: Текстурирование и Координаты (/TextureSampling)
**Цель:** Понимание UV-развертки и работы растеризатора.
- [x] Вывести текстуру на треугольник (использовать свою уникальную текстуру).
- [x] Визуализация координат в отчете:

- [x] Вывести UV-координаты как цвет.

### Задание 3: Матричные преобразования
**Цель:** Изучение конвейера трансформаций и работы с матрицами в вершинном шейдере.
- [ ] **Передача матриц:**
    - Разобраться, как задаются и передаются отдельные матрицы из FX Composer в шейдер.
- [ ] **Пошаговая трансформация:**
    - Реализовать в вершинном шейдере последовательное умножение координат вершины на каждую матрицу отдельно.
    - Получить промежуточные результаты (переход из Local Space в World, затем в View и Clip Space).
- [ ] **Объединенная трансформация:**
    - Реализовать умножение вершины на одну предвычисленную матрицу `WorldViewProj`.
- [ ] **Сравнение:**
    - Убедиться, что визуальный результат последовательного умножения и умножения на одну итоговую матрицу идентичен.
